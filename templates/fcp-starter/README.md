# fcp-starter

Scaffold for building a new FCP server (TypeScript).

## Quick Start

1. Copy this template: `cp -r templates/fcp-starter fcp-yourdomain`
2. Find and replace `yourdomain` with your domain name
3. `npm install`
4. Implement your semantic model in `src/model/`
5. Register verbs in `src/server/verb-registry.ts`
6. Implement serialization in `src/serialization/`
7. `npm test` and `npm run build`

See [Writing an FCP Server](../../docs/writing-an-fcp-server.md) for the full guide.
